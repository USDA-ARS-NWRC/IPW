Functional Changes in Version 2

Hydro-Climatic Modeling Utilities

Version 2 includes about 200 commands, with over 120 of these directly related to hydro-climatic modeling and analysis. IPW commands are grouped into functional categories as follows:

  • Environmental Physics and Properties: Point
  • Environmental Physics and Properties: Spatial
  • Physical Modeling: Point
  • Physical Modeling: Spatial
  • Radiation Modeling: Point
  • Radiation Modeling: Basin-Scale
  • Radiation Modeling: Global-Scale
  • DEM Processing
  • Geographic Projection
  • Format Conversion
  • Data, and Image Import/Export
  • Basic Data Processing
  • Image Query
  • Image Computation
  • Image Filters, Orientation, Sampling
  • Image Header Manipulation
  • Band Combination and Separation
  • Image Classification and Transformation
  • Miscellaneous

Integration of Point Processing into IPW

Procedures for processing point data have been fully integrated into Version 2 including the integration of library functions and procedures for developing point utilities. Point processing programs can be used to test and develop algorithms and solutions which numerically represent physical processes. These can then be easily converted to a spatial context, using the same library routines.

Integration of IPW and Physical Modeling Libraries

Integrated IPW and physical modeling libraries developed by USGS. Separated these into:

  • libipw -- contains all library functions related to IPW images
  • libmodel -- contains all the functions for physical modeling

Added several new libraries:

  • libsnobal -- functions for energy balance snowmelt modeling
  • libgctp -- functions from the General Cartographic Transformation Package
  • libtopo -- functions for topographic calculations
  • libimage -- functions for higher-level interface to IPW images
  • libstropts -- functions for using string options for commands

For a detailed list of all changes and additions included in IPW Version 2 see: What's New.

Image Display Utilities

Two image display utilities were modified to interpret and display IPW images. xli Version 1.6 by Graeme Gill, which was derived from xloadimage, and xv Version 3.10a by John Bradley.

  • xloadimage+ipw.tar.Z

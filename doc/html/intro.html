The software described in this electronic document was developed as a joint effort between the USDA Agricultural Research Service and the US Geological Survey, Water Resources Division. It is currently under review to be released as an ARS Technical Report, and a USGS Open File Report.

This electronic document contains the complete description of the Image Processing Workbench (IPW), ARS - USGS Version 2 software toolkit, including a description and history, installation and configuration instructions, and both the User's and Programmer's Guides, with manual pages for all commands, scripts, and library routines. It was designed from the outset as a hypertext document, to be viewed with a web browser.

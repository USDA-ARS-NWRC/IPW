IPW Programmer's Guide

This guide assumes the reader is familiar with the C programming language, the Perl language, and the UNIX make utility. IPW uses GNU's version of make for its additional functionality such as include files, built-in functions, and conditionals which simplifies the management and use of Makefiles in IPW's source code.

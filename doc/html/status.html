Status of IPW Version 2

22 May 1998
Initial draft of Programmer's Guide completed.

10 Mar 1998
Initial draft of User's Guide completed.

30 Jan 1998
Converted coding standard from Frew's thesis into a HTML document.

15 Jan 1998
Completed documentation for new installation procedure.

1 Dec 1997
Tested new installation procedure.

16 Oct 1997
Began using beta version of IPW v2.0 here at NHEERL-WED.

14 Oct 1997
Ported the library for energy-balance snowmelt model (snobal). Completed the port of all IPW commands.

26 Sep 1997
Completed the port of ERLC library. Physical modeling functions now reside in a separate IPW library. IPW commands remaining to be ported are those that were originally part of QDIPS package.

14 Aug 1997
Completed the port of the GCTP library. The port of the ERLC library is almost complete. All the functions ported so far have been integrated into version 2 of the standard IPW library. The remaining functions are physical modeling functions that were originally part of the QDIPS package. Also initiated the port of the additional IPW commands developed here (ERLC commands).

18 Jul 1997
Completed the port of the base IPW library to version 2. Initiated the port of the base IPW commands. Also initiated the port of the additional IPW libraries developed here at NHEERL-WED (formerly ERLC) for physical modeling and the General Cartographic Transformation Package (USGS's GCTP).

17 Jul 1997
Generated the "configure" script using "autoconf" utility.

26 Jun 1997
Converted the manual pages for all user commands to version 2 format.

29 May 1997
Converted the text file with the installation instructions into the Installation Guide.

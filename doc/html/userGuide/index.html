IPW User's Guide

This guide assumes that the new IPW user is already familiar with UNIX as a user.

Data Files

IPW data files are simply UNIX files. There are two types of data files:

  • text files that contain a sequence or series of data records, and

  • image files that contain spatial data for a geographic region.

IPW commands

Each IPW command is a separate UNIX command. The syntax for using an IPW command follows UNIX conventions: 

     command [ options ] [ operands ]
Like other UNIX commands, each IPW command has a manual page that describes what the command does, its options and operands. The manual pages for the IPW commands are organized in two different ways:
  • Grouped into categories based on their functionality, and

  • Indexed alphabetically by their names and associated keywords.
